About this Component

This experiment demonstrates a hybrid rendering approach:

  • WebGL2 Layer: Renders the static "data core" geometry (wireframe icosahedron) and the scanning grid floor. Uses standard vertex/fragment shaders.
  • WebGPU Layer: Handles the dynamic "data stream" particles using compute shaders for physics simulation (attraction/repulsion to the core) and high-performance instanced rendering.
  • CSS Layer: Provides the UI overlay and status indicators.

The two contexts share the same coordinate space but are rendered on separate, perfectly aligned canvases.
