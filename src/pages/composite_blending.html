Implementation Details

This demo combines multiple rendering contexts layered via CSS:

  • Backing Layer (Bottom) - Optional 2D canvas providing mid-gray backing for overlay/soft-light blend modes to work correctly.
  • WebGPU (Middle) - Renders the "Circuit" background with compute shaders.
  • WebGL2 (Top) - Renders the "LED/Lens" foreground with instanced rendering.
  • CSS Blend Modes - Controls how canvases composite (Hard Light, Screen, Overlay, etc).

Why Overlay Was Invisible

The overlay blend mode multiplies colors when the base is dark (<0.5) and screens when light (>0.5). With the original dark background (#050505 ≈ 0.02), overlay multiplied everything to near-black. The fix adds a mid-gray backing layer that provides values around 0.5, allowing overlay to properly blend.
