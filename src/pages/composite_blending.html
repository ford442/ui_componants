Implementation Details

This demo combines two rendering contexts layered via CSS:

  • WebGPU (Bottom) - Renders the "Circuit" background with compute shaders.
  • WebGL2 (Top) - Renders the "LED/Lens" foreground with instanced rendering.
  • CSS Blend Modes - Controls how the WebGL2 canvas composites over the WebGPU canvas (Hard Light, Screen, etc).
